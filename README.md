# librarian

---

[electron](https://electron.atom.io/)で作成した伺か的な何かです。  
簡易英訳&辞書アプリ

## Install
* `$ git clone https://github.com/marefa9cb8/librarian.git`
* `$ cd librarian`
* `$ npm install`
* `$ ./node_modules/.bin/electron .`

## Usage
* 検索ワードを入力するとリアルタイムで英訳及び辞書検索が行われて表示されます。  
  - Translation → 英訳結果
  - Dictionary → 辞書結果
